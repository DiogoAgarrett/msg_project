# Quick Start Guide

## Prerequisites
- Java 21+
- Node.js 18+
- Maven

## Backend (Spring Boot)

### 1. Navigate to backend directory
```bash
cd msg_exercicios
```

### 2. Run the application
```bash
mvn spring-boot:run
```

**Alternative (IntelliJ/IDE):**
- Open `src/main/java/exercicio2/Main.java`
- Click the green play button ▶️

**Backend will start on:** `http://localhost:8080`

---

## Frontend (React + Vite)

### 1. Navigate to frontend directory
```bash
cd frontend
```

### 2. Install dependencies (first time only)
```bash
npm install
```

### 3. Run the application
```bash
npm run dev
```

**Frontend will start on:** `http://localhost:5174`

---

## Access the Application

1. **Open browser:** `http://localhost:5174`
2. **Select a book** from the dropdown
3. **Select an edition** from the second dropdown
4. **View the table of contents**

---

## Stopping the Applications

- **Backend:** Press `Ctrl+C` in the terminal
- **Frontend:** Press `Ctrl+C` in the terminal
